## Problem Statement

Given an array of integers, find the minimum and maximum elements present in the array.

### Example

Input:
[3, 2, 1, 56, 10000, 167]

Output:

[1, 10000]


Approach
Initialize:
max with Integer.MIN_VALUE
min with Integer.MAX_VALUE
Traverse the array using a for-each loop.
Update minimum and maximum values accordingly.
Store results in an ArrayList.
